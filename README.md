# Andx-Editor
Andx Editor 是 ANDX CMS 项目默认的HTML编辑器，满足基础文本编辑功能，总共只有几十KB，加载和运行速度快，适合研究学习、对编辑环境要求不高的用户使用。如：网站后台备注等文本框使用，网站前端替换UBB编辑器(请做好对应的服务器端程序处理防止出现安全问题)等。<br />
